TestZilla

Welcome to TestZilla, An open source & distributed solution for stress test and benchmarking on Application Services, APIs, Web Servers and Network Protocols. TestZilla is currently under development, and it's initial version is focused on Web Application Services and API. At TestZilla development, we are trying to provide you with a home-based and cost-effective solution for setting up a test lab. Currently, there are many open source solutions for stress testing and benchmarking in the open source community, thanks all of them. We are trying to provide all the functional features of these tools in the form of unique solution.
